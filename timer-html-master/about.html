About This Project

Data Used:

I used NYC open data for my project and specifically used the open positions data set to display open jobs by department. On the first screen users can select a department to look at. The second screen which appears shows all positions open for that deparment. The third screen would show more detail on a specific job based on job number Link: https://data.cityofnewyork.us/City-Government/NYC-Jobs/kpav-sd4t

What Framework?

I found a bootstrap theme from a website, the theme is titled: Timer and can be found at: https://themewagon.com/themes/timer-free-responsive-multi-page-personal-bootstrap-template/ I chose this particular theme because we have worked with things simalar to this before but this was differnt in a variety of ways and I thought would challenge me. It turned out to challenge me quite a bit.

Process

This took me approximately 18 hours of work to accomplish. I first decided to find an API to work with. I then moved onto finding an actual bootstrap template which would serve as my layout. I then had to figure out how to use forms and redirecting to seperate pages, which my template provided some background with. The most challenging part was from page 2-3, trying to get info from a table clickable and easy to move over to the next page.
